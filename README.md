Emotion Analysis in Text (NLP Project)

 Overview
This project analyzes emotional patterns in text data using Natural Language Processing (NLP) and data visualization techniques.

 Dataset
- Emotion-labeled text dataset with 6 categories:
  joy, sadness, anger, fear, love, surprise
- ~16,000 text samples

Techniques
- Text preprocessing (cleaning, normalization)
- Feature extraction (word count, text length, unique words)
- Exploratory Data Analysis (EDA)
- Visualization of linguistic patterns

 Key Insights
- Joy and sadness are the most frequent emotions
- Text length and word count show strong correlation
- Different emotions exhibit distinct linguistic patterns

 Tools
Python, pandas, NumPy, NLTK, matplotlib, Power BI

 Visualization
Interactive dashboards were created using Power BI to explore emotion distributions and patterns.
